API Documentation

Retrieve the homepage of the Smart Web Stream API.

Parameters: None

Response: HTML content of the homepage

curl -X GET "http://147.93.19.133:8000/"

Stream a Telegram file by its ID with code-based authentication.

Parameters: file_id (integer, required), code (string, required)

Response: HTML player page with streaming URL

curl -X GET "http://147.93.19.133:8000/stream/12345?code=abc123"

Download a Telegram file by its ID with code-based authentication.

Parameters: file_id (integer, required), code (string, required), Range (header, optional)

Response: File stream (partial if Range header provided)

curl -X GET "http://147.93.19.133:8000/dl/12345?code=abc123" \
-H "Range: bytes=0-1048575"
